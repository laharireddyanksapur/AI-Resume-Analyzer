# AI Resume Analyzer

AI Resume Analyzer is a full stack web application that analyzes resumes using AI and provides ATS score, skill analysis, improvement suggestions, and resume feedback.

## Features

- Resume upload system
- AI powered resume analysis
- ATS score generation
- Skill extraction
- Resume improvement suggestions
- Dashboard interface
- Responsive UI
- Frontend and backend integration
- OpenAI API integration

## Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### AI Integration
- OpenAI API
